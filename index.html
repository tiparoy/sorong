BLOGGER GALLERY CREATOR

Ini adalah project pertama saya dalam belajar javascript, CSS, dan HTML. Tools ini hanya membantu anda membuat foto gallery di postingan blogger.
Cara menggunakannya adalah sebagai berikut:

  1. Buat Postingan baru di blogger
  2. Masuk pada mode/tampilan HTML
  3. Upload seluruh foto/gambar yang akan ditampilkan
  4. Salin teks HTML dari gambar yang diupload tersebut
  5. Paste pada area input yang tersedia dibawah
  6. Klik "Convert"
  7. Copy Paste teks HTML hasil konvert ke blog anda dalam mode HTML
  8. Edit teks Judul dan keterangan gallery
  9. Klik Pratinjau untuk melihat hasil
